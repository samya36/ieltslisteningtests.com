# 雅思听力评分标准解析网站

## 项目简介
本网站旨在为雅思考生提供全面、专业的听力评分标准解析，帮助考生更好地理解评分机制，提高备考效率。

## 功能特点
1. 评分机制解析
   - 40道题目的分值分布说明
   - 完整的分数转换对照表
   - 各分数段特点分析

2. 互动功能
   - 分数计算器
   - 目标分数规划器
   - 用户反馈表单

3. 教学资源
   - 真实考生案例分析
   - 题型策略指导
   - 免费练习题

## 项目结构

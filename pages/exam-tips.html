模考前必须知道的 7 个得分细节

每个细节值 1–2 分，7 个加起来就是半个分数段

雅思听力的很多丢分不是因为"没听懂"，而是因为答题细节没做对。以下 7 个细节是模考和实考中最高频的丢分点——考前过一遍，考场上少丢 3–5 分完全可能。

读题速度：30 秒预览一组题

每个 Section 开始前有大约 30 秒的读题时间。很多考生把这 30 秒浪费在"看题但没看进去"上。

正确做法：30 秒内完成三件事——①看清题型（填空 / 选择 / 匹配）②圈出每题的关键词 ③预判答案类型（名词 / 数字 / 日期）。
错误做法：逐字逐句读题目，读到一半音频就开始了，手忙脚乱。

练习方法：拿一套新题，不听音频，只练读题。计时看自己能不能在 30 秒内完成 5 道题的关键词标注。

单复数：一个 s 值一分

填空题的答案如果需要复数形式但你写了单数，或者反过来，这道题就是 0 分。

题目：The course covers several different       .
答案：topics（复数）。如果写 topic → 0 分。

判断方法：看空格前面的词。如果有 "several / many / various / different / two / three"，答案一定是复数。如果有 "a / an / one / each"，答案是单数。

额外注意不规则复数：child → children, person → people, criterion → criteria, phenomenon → phenomena。

数字和日期拼写

Section 1 几乎必考数字和日期。常见的丢分方式：

  • 电话号码：double 6 = 66，不是 26。oh = 0
  • 日期格式："the 3rd of March" 写成 3 March 或 3rd March 或 March 3 都可以，但不能写 3/3 或 03/03
  • 金额：$3.50 要写清楚小数点。"three fifty" 可能是 3.50 也可能是 350，需要根据上下文判断
  • 时间："a quarter past two" = 2:15，"half past ten" = 10:30
练习建议：每天花 10 分钟做数字听写练习。念一串电话号码、日期、金额，写下来对答案。这是最容易提分的基本功。

转折词修正：答案在 but 后面

这可能是雅思听力最经典的陷阱模式：说话人先给一个信息，然后用转折词否定或修正。

"The meeting was originally on Tuesday... but actually, it's been changed to Thursday."

答案 = Thursday，不是 Tuesday。

需要警觉的转折词和修正信号：

  • but / however / although / actually / in fact
  • sorry, I meant... / no, wait... / oh, I was wrong...
  • it's been changed to... / the updated one is...
提醒：写答案时先轻写、留空位。等这道题的对话完全结束后再确认最终答案。不要听到第一个信息就写死。

大小写：不扣分但别冒险

官方说法是大小写不影响评分——"monday" 和 "Monday" 都算对。但有两个注意点：

  • 专有名词建议大写：人名、地名、月份、星期。虽然小写不扣分，但大写显示你知道这是专有名词，也减少阅卷歧义
  • 全大写也可以：如果你不确定，全部大写写（如 MONDAY）是最安全的策略
实用策略：如果是纸笔考试，所有答案用大写字母写，彻底避免大小写问题。如果是机考，按正常规则写即可。

字数限制：多一个词 = 零分

每组填空题都会标注字数限制，常见的有三种：

  • ONE WORD ONLY — 只能写 1 个词。"car park" 算两个词，不符合
  • NO MORE THAN TWO WORDS — 最多 2 个词
  • ONE WORD AND/OR A NUMBER — 可以是 1 个词，或 1 个数字，或"词 + 数字"组合（如 "floor 3"）

要求 ONE WORD ONLY，原文说 "the swimming pool"
答案写 "pool" ✓   写 "swimming pool" ✗（两个词，超出限制）

常见失误：忘记看字数限制，凭感觉写了两三个词。每次做填空题之前，先把字数限制在题目上圈出来。

拼写准确：差一个字母就是零分

雅思听力对拼写的要求是 100% 准确。以下是考生最常拼错的词：

accommodation ✓
accomodation ✗
Wednesday ✓
Wendsday ✗
February ✓
Feburary ✗
library ✓
libary ✗
environment ✓
enviroment ✗
government ✓
goverment ✗
restaurant ✓
restarant ✗
maintenance ✓
maintainance ✗
receipt ✓
reciept ✗
建议：考前一周，把上面这些词 + 你个人错题本里的拼写错误集中过一遍。写 3 遍比看 10 遍有效。

7 个细节全掌握，少丢 3–5 分不是问题。
